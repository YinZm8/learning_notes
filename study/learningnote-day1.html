第五节 标题和段落

    1.作用
        一般用在新闻标题、文章标题、网页区域名称、产品名称等等

        h1～h6（双标签）

        文字加粗&自号逐渐减小&独占一行

    ——段落标签
    1.作用
        一般用在新闻段落、文章段落和产品描述信息等等
    2.标签名
        p（双标签）
    3.显示特点
        -独占一行
        -段落之间存在间隙
        -网页文本自动适配换行

第八节 图像标签

    1.作用
        在网页中插入图片
    2.代码样式
        <img src="图片的URL">
    3.src作用
        src用于制定图像的位置和名称，是<img>的必须属性
    4.图像标签的属性
        属性    作用          说明                          备注
        alt    替换文本    图片无法显示的时候显示文字      以前网速慢，可能导致图片加载不出来，不想影响内容的浏览，用alt替换
        title  提示文本    鼠标悬停在图片上的是显示的文字
        width  图片的宽度  值为数字，没有单位             浏览器缩放图片，默认是等比例缩放
        height 图片的高度  值为数字，没有单位
    注：与其他属性用空格隔开 如- <img src="url" alt="替换文字">

第九节 相对和绝对路径

    1.路径定义
        路径指的是查找文件是，从起点到终点的路线
    2.路径的分类
        相对路径：从当前文件位置出发查找目标文件
        绝对路径：从根目录（盘符）出发查找目标文件
    3.方式
        ——相对路径
        使用方式：/ 表示进入某个文件夹里面； ./ 表示进入当前所在文件夹； ../ 表示进入当前文件夹的上一级文件夹； ../../ 就是进入两级上级目录，再次上级也是以此类推；
        应用场景：找自己的电脑的文件
        ——绝对路径
        使用方式：/根目录/其他目录/...等 注：按下 Command + Option + C 键，可以快速复制文件的绝对路径
        应用场景：友情链接（文件的在想网址）

第十节 超链接标签

    1.作用
        点击到其他页面
    2.代码样式 
        <a href=""></a>   注：href属性值是跳转地址，是超链接的必须属性
    3.使用方式
        可以跳转到外部链接，也可以跳转到本地文件；空格加上属性target="_blank"，新窗口跳转页面————样式：<a href="跳转网址/本地路径" target="_blank">页面展示名称</a>
    经验分享：开发初期，不知道超链接的跳转地址，href属性值写#，表示空链接，不会跳转

第十一节 多媒体标签-音频和视频

    ——音频标签
    1.音频标签代码
        <audio src="音频的url"></audio> 
    2.常见属性
        属性             作用                特殊说明
        src （必须属性） 音频URL           支持格式：mp3，ogg，wav
        controls       显示音频控制面板    在html5中，如果属性名和属性值完全一样，可以写为一个单词（布尔变量不需要写属性值）
        loop           循环播放     
        autoplay       自动播放           为了提升用户体验，浏览器一般会禁用自动播放功能
    ——视频标签
    1.视频标签代码
        <video src=""></video>
    2.常见属性
        属性             作用                特殊说明
        src （必须属性） 视频URL           支持格式：mp4，ogg，webm
        controls       显示视频控制面板    在html5中，如果属性名和属性值完全一样，可以写为一个单词（布尔变量不需要写属性值）
        loop           循环播放   
        muted          静音播放
        autoplay       自动播放           为了提升用户体验，浏览器只支持在静音状态自动播放
